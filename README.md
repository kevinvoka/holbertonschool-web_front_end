# Holberton School — Web Front End

![Holberton](https://img.shields.io/badge/Holberton-School-red?style=flat-square) ![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat-square&logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat-square&logo=css3)

## Description

This repository covers **HTML and CSS** from the ground up — building the foundations of modern web development. Projects range from semantic HTML structure to advanced CSS layouts using Flexbox and Grid.

## Projects

| Directory | Topic |
|-----------|-------|
| `html_basic` | HTML structure, tags, forms |
| `html_advanced` | Semantic HTML5, accessibility |
| `css_basic` | CSS selectors, box model |
| `css_advanced` | Flexbox, Grid, responsive design |
| `sass_scss` | SASS/SCSS preprocessor |
| `flexbox` | Advanced Flexbox layouts |
| `Bootstrap` | Bootstrap grid and components |
| `accessibility` | WCAG guidelines, ARIA roles |
| `responsive_design` | Mobile-first, media queries |

## Learning Objectives

- Write valid, semantic HTML5
- Style pages with CSS3 using Flexbox and Grid
- Build responsive layouts that work on all screen sizes
- Use SASS/SCSS to write maintainable stylesheets
- Follow accessibility best practices (WCAG)
- Integrate Bootstrap for rapid UI development
